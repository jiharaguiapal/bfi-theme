# bfi-rectrack-style

## Install

```
npm install bfi-style
```

```
Note that installing as a dev dependency --save-dev or -D will not work correctly.

```

### Usage

```
Go to nuxt.config.js
Add to global css

    css: ["./node_modules/bfi-theme/scss/style.scss"],
    plugins: [],


```
